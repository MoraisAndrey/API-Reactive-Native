# Pokedex React Native

Aplicação desenvolvida em React Native utilizando Expo que consome dados da PokeAPI.

## 🚀 Tecnologias utilizadas

- React Native
- Expo
- Expo Router
- Firebase
- PokeAPI
- Vercel (Deploy)

## 🌐 Aplicação online

Acesse a aplicação:

https://pokedexreactnative.vercel.app/

## 🔗 API utilizada

https://pokeapi.co/

## 🔥 Funcionalidades

- Listagem de Pokémons
- Pesquisa de Pokémon
- Página de detalhes do Pokémon
- Favoritar Pokémon (Firebase)
- Rotas dinâmicas

## 🏗 Arquitetura da aplicação

App React Native (Expo)
↓
Consome dados da PokeAPI
↓
Salva favoritos no Firebase
↓
Deploy no Vercel

## Instalação do projeto

Clone o repositório:



Entre na pasta do projeto:
cd pokedex-react-native


Instale as dependências:
npm install

Execute o projeto:
npx expo start
